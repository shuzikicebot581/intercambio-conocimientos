# Intercambio de Conocimientos

## Descripción
Intercambio de Conocimientos es una aplicación web diseñada para ayudar a desarrolladores hispanohablantes a compartir recursos, tutoriales y experiencias dentro de la comunidad de desarrollo.

## Funcionalidades
- Registrarse e iniciar sesión
- Crear y compartir tutoriales
- Comentar y valorar recursos compartidos
- Sistema de búsqueda y filtros por categorías
- Notificaciones de nuevos recursos y comentarios

## Instalación
Para ejecutar el proyecto localmente, sigue estos pasos:
1. Clona el repositorio: `git clone https://github.com/tu-usuario/intercambio-conocimientos`
2. Navega al directorio del proyecto: `cd intercambio-conocimientos`
3. Instala las dependencias: `npm install`
4. Ejecuta el servidor: `npm start`
